# Projeto Agrinho 2026 - "Agro forte, futuro sustentável."
# Sobre:
O Agro Forte, Futuro Sustentável é uma plataforma web dedicada a conscientizar e educar sobre o equilíbrio essencial entre a alta produtividade agrícola e a preservação ambiental. Com um design limpo e focado na legibilidade, o site aborda os desafios da segurança alimentar global, destaca o papel dos Sistemas Agroflorestais (SAF) como maquetes vivas para o aprendizado e traz um toque visual único através de ilustrações personalizadas em pixel art. Um espaço digital informativo que conecta tecnologia, educação e sustentabilidade para o futuro do campo.
